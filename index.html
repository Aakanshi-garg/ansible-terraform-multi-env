DevOps Project: Infra Automation

Terraform + Ansible + AWS | Fully Automated Stack

🚀 Project Overview

This project automates the provisioning of cloud infrastructure using Terraform and configures servers using Ansible. It's designed to accelerate deployment, improve consistency, and reduce manual effort.

🧱 Tech Stack

  • 💠 Terraform for Infrastructure as Code
  • 🔧 Ansible for Configuration Management
  • ☁️ AWS EC2, VPC, S3, and more
  • 📜 YAML + Bash + HTML

📦 Sample Terraform Snippet

resource "aws_instance" "my_instance" { ami = "ami-12345678" instance_type = "t2.micro" key_name = "infra-key" }

📂 Project Structure

/terraform-custom-modules/ ├── infra-app/ │ ├── ec2.tf │ ├── s3.tf │ └── dynamodb.tf ├── remote-infra/ ├── infra-key └── main.tf
